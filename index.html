引き継ぎ重要事項

最初に読むべき事項
総会・運営
市P総会は3月開催に変更すべき
単Pの引き継ぎ繁忙期（3〜4月）に市P総会が重なると出席・集中が難しい。3月上旬前倒しで単Pの新旧引き継ぎ前に市Pの意思決定を完了できる。
▶ 後任へのアドバイス
年度当初から次年度総会日程を3月で仮押さえすること。変更には役員会の合意が必要。
年間スケジュール設計
4月・5月初旬に分会長会を2回設定
単P新会長への説明・サポート・出向委員会の役割決めを5月下旬の単P総会に間に合わせるには4月と5月初旬の2回が必須。
▶ 後任へのアドバイス
年度初めに会場確保・日程調整を最優先で。単P会長は就任直後で多忙なため連絡は丁寧に・早めに。
単P支援
年度当初の単P支援が最も密度が高い時期
年度が明けると①市Pの役割説明②立ち上げ期の案内③単P総会で決めるべき事項の共有④活動方針相談⑤出向委員会の役割決め、が目白押し。この時期の動きが1年間の関係構築を左右する。
▶ 後任へのアドバイス
市P役員は引き継ぎが軽微（継続メンバーが主軸）なので4月の単P支援に全力を向けられる。この時期を惜しまないこと。

使い方・更新方法

GitHub自動更新・日常の記録手順
⚡ GitHub Pages 自動更新

GitHubのPATとリポジトリ情報を入力しておくと、Claudeがこのファイルを更新するだけで自動的にWebに反映されます。入力値はこのブラウザにのみ保存されます。

注意：PATはパスワードと同等です。スコープは必ず public_repo のみ に絞ってください。このページはあなた自身のブラウザにのみ保存されます。他人のPCで使わないでください。
GitHub Pages 初期設定手順
1
PATを発行： github.com/settings/tokens/new → Note: pta-hikitsugi → スコープ: public_repo のみ → Generate token → コピー保存
2
リポジトリ作成： github.com/new → 名前: pta-hikitsugi → Public → Add README → Create
3
初回ファイルアップロード： リポジトリ → Add file → Upload → index.html をドロップ → Commit changes
4
Pages有効化： Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save
5
URL確認（数分後）： https://（ユーザー名）.github.io/pta-hikitsugi/ にアクセスできれば完成。このURLを単P会長に共有。
6
以降の更新：上の設定欄にPATを入力 → Claudeに報告 → 「反映」ボタン → Claudeが新しいHTMLを出力 → 同じ手順で上書きアップロードするだけ。
日常の使い方
1
月ごとに新規チャットを開く
プロジェクト内で 「2025年4月」 のようにチャットを立てる。開くと自動でテンプレートが表示される。
2
活動報告・気づきを話しかける
テンプレートに沿って入力するか話し言葉でOK。Claudeが聞き返しながら整理。資料（PDF・Word等）はそのまま添付。
3
「現況を反映」ボタンで資料を更新
右下の赤いボタン → メッセージをコピー → Claudeに送る → 新しいHTMLが出力される → 同リポジトリに上書きアップロード → URL自動反映。
4
月末に月次まとめをナレッジに保存
「今月の記録を要約して」と送り、要約をプロジェクトのナレッジに追加。次月は短いチャットで済む。

後任会長へ

このツールと引き継ぎの使い方
このHTMLについて
1
このHTMLは引き継ぎ資料そのものです
1年間の活動日報・気づきメモ・会議記録がすべて蓄積されています。スマホ・PCどちらからでも閲覧できます。URLは前任会長に確認してください。
2
Claudeプロジェクトを引き継ぐ
前任会長のClaude プロジェクト「交野市PTA協議会 会長引き継ぎノート」の指示文をコピーして、自分のClaudeアカウントで同名プロジェクトを作成してください。指示文は前任会長から受け取るか、このHTMLの「月次テンプレート」欄を参照してください。
3
GitHubリポジトリを引き継ぐか新規作成する
前任会長のリポジトリに書き込み権限をもらう（Collaborator追加）か、自分のGitHubに新しいリポジトリを作って最新のHTMLをアップロードすれば、自分のURLで公開できます。
4
2月から記録を始める
着任準備月（2月）からClaudeに話しかけて記録を積み上げてください。前任者の1年分の記録がこのHTMLにすべて残っています。月別詳細を読むと流れがわかります。
